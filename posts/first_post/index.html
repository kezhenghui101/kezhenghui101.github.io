Hugo+Github Pages

  • 把模版部署在网站上
  • hugo –theme=LoveIt –baseURL=“https://kezhenghui101.github.io/" -D
  1. 生成public文件夹，然后把文件夹push至GitHub
  2. cd public
  3. git init
  4. git add .
  5. git commit -m “first”
  6. git remote add origin git@github.com:kezhenghui101/kezhenghui101.github.io.git
  7. git push -u origin main ssh-keygen -t rsa -C “kezhenghui101@outlook.com”

测试

public文件夹与GitHub建立连接即可

每次新增加文章后，hugo –theme=LoveIt –baseURL=“https://kezhenghui101.github.io/" -D

然后 git add commit push一条龙

后续添加文章过程

  1. hugo new posts/test.zh-cn.md
  2. 写文章或者更改之前的文章
  3. hugo serve -D 本地渲染看下效果如何
  4. hugo -D
  5. npm run algolia 上传索引文件
  6. cd public
  7. git add .
  8. git commit -m “message”
  9. git push
